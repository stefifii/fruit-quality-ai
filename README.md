# 🍎 Fruit Quality Detection AI

An AI-powered web application that detects whether a fruit is fresh or rotten using Deep Learning and Computer Vision.

---

# 📌 Features

* Upload fruit images
* Webcam image capture
* Fresh vs Rotten prediction
* Streamlit web interface
* Deep Learning image classification

---

# 🧠 Technologies Used

* Python
* TensorFlow
* Keras
* OpenCV
* Streamlit
* NumPy
* Pillow

---

# 📂 Dataset Classes

* Fresh Apple
* Rotten Apple
* Fresh Banana
* Rotten Banana

---

# ⚙️ How It Works

1. Upload a fruit image
2. Image is preprocessed
3. CNN model analyzes the image
4. Prediction result is displayed
